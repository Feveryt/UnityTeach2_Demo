# 弹幕射击 STG · Bullet Hell Shooter

> Unity 2D 飞机大战 | 核心开发 | 

![Gameplay](Demo/demo.gif)

## 项目简介

一款纵版弹幕射击游戏，玩家操控飞机在屏幕底部移动，躲避并击破来自屏幕各方位发射的密集弹幕。项目实现了完整的游戏循环：选角、战斗、排行榜、设置等模块。

**本人主导的优化与设计**：
- 基于 Unity 内置 `ObjectPool<T>` API 构建全局子弹与特效池，消除频繁 Instantiate/Destroy 带来的 GC 压力。
- 设计 XML 配置表驱动子弹弹道（5 种运动模式）与发射波次，实现数据与逻辑分离。
- 封装泛型单例面板基类 `BasePanel<T>`，统一 UI 生命周期管理。
- 通过 `GameDataMgr` 集中管理音乐、音效、排行榜等数据的存取与本地持久化。

## 游戏功能

- 角色 WASD 移动，边界限制与倾斜动画反馈
- 8 方向屏幕边缘弹幕发射，支持扇形与序列波次
- 5 种子弹运动模式：直线、正弦曲线、左右旋转、追踪导弹
- 子弹碰撞伤害、血量管理、死亡结算
- 主菜单、选角界面、设置面板（音量/音效调节）
- 排行榜系统，支持成绩录入与排序
- 音乐音效全局管理，数据本地持久化
